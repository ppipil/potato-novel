## 今天遇到的问题与处理

### 1. skill 引用文档不可用

现象：

- 官方 `skill.md` 能获取
- 但 `references/*.md` 下载结果是网站 HTML 壳

影响：

- 无法完整依赖 skill 内部的 reference 指南

处理：

- 改为结合项目实际代码、官方文档站、平台官网前端资源来反推关键字段

### 2. 前端依赖安装失败

现象：

- `npm install` 先出现依赖树冲突
- 后续又出现全局 npm cache 权限问题

处理：

- 将 Vite 版本调整到稳定兼容范围
- 改用工作区本地 npm cache 安装

### 3. FastAPI 启动报 Python 3.9 类型注解错误

现象：

- `dict[str, Any] | None` 在 Python 3.9 报错

处理：

- 改为 `Optional[Dict[str, Any]]`

### 4. 后端持续 500，提示缺失配置

现象：

- `/api/me` 和 `/api/auth/login` 返回 500

原因：

- `.env` 加载路径依赖启动目录
- shell 变量可能覆盖 `.env`
- `SECONDME_CLIENT_SECRET` 一度仍是占位值

处理：

- 固定读取 `backend/.env`
- `load_dotenv(..., override=True)`
- 检查真实读入值

### 5. 登录成功后仍回首页

现象：

- OAuth 成功
- 但 `/api/me` 返回 `{"authenticated": false}`

原因：

- 早期 session cookie 里塞入了完整 token 和完整 user info
- cookie 体积过大，浏览器未实际保存

处理：

- 改为 cookie 中只存最小会话信息
- token 留在服务端内存

### 6. integration 仍无法真实提交

现状：

- 本地 manifest 和 MCP endpoint 已完成
- 但平台 validate 仍缺公网地址

原因：

- `localhost` 不能作为最终 integration 校验地址

处理：

- 明确当前阻塞为“需要部署到公网”
